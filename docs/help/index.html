Overview / Aperçu

English

ScreenVoiceRec captures audio on your Mac and saves it as a file you can play back or share. You can record:

  • System audio from an entire display (desktop audio)
  • Audio from a single application window (e.g. a browser tab playing video)
  • Input from a microphone, headset mic, or line-in (3.5 mm jack / USB audio interface)

The main window is divided into four areas (numbered in the screenshot above).

Français

ScreenVoiceRec capture l’audio sur votre Mac et l’enregistre dans un fichier que vous pouvez écouter ou partager. Vous pouvez enregistrer :

  • L’audio système d’un écran entier (audio du bureau)
  • L’audio d’une seule fenêtre d’application (p. ex. un onglet de navigateur)
  • L’entrée d’un micro, d’un micro-casque ou d’une entrée ligne (prise 3,5 mm / interface USB)

La fenêtre principale comporte quatre zones (numérotées sur la capture d’écran ci-dessus).

1

Source

English

Choose what to record.

  1. Source type (top menu): pick one of
    • Entire Display (Desktop Audio) — all system audio on the selected monitor
    • Single Window — only the audio of one window (shown in the screenshot)
    • 3.5 mm Headphone Jack / Line In — hardware input (mic or line-in)
  2. Second menu (when applicable): select the target display or window. Window entries show App name — Window title (e.g. Google Chrome — YouTube).
  3. Refresh Windows & Displays — reloads the list after you open, close, or move windows.
Tip: Start playback in the target app before recording so the level meter confirms audio is flowing.

Français

Choisissez quoi enregistrer.

  1. Type de source (menu du haut) :
    • Écran entier (audio du bureau) — tout l’audio système de l’écran choisi
    • Fenêtre unique — uniquement l’audio d’une fenêtre (comme sur la capture)
    • Prise casque 3,5 mm / entrée ligne — entrée matérielle (micro ou ligne)
  2. Deuxième menu (si applicable) : choisissez l’écran ou la fenêtre cible. Les fenêtres s’affichent sous la forme Nom de l’app — Titre de la fenêtre.
  3. Actualiser fenêtres et écrans — met à jour la liste après ouverture, fermeture ou déplacement de fenêtres.
Astuce : Lancez la lecture dans l’application cible avant d’enregistrer, afin que le niveau d’entrée confirme la présence du signal.
2

Export Format / Format d’exportation

English

Select the file format before you start recording. Available formats:

  • AAC (M4A, lossy) — small files, good for sharing (default in screenshot)
  • Apple Lossless (M4A, lossless) — no quality loss, larger than AAC
  • FLAC (CAF container, lossless) — lossless compression in a Core Audio Format file
  • Linear PCM (WAV, uncompressed) — maximum compatibility for editing

The extension is chosen automatically (.m4a, .caf, or .wav).

Français

Choisissez le format de fichier avant de commencer l’enregistrement. Formats disponibles :

  • AAC (M4A, avec perte) — fichiers compacts, pratiques pour le partage
  • Apple Lossless (M4A, sans perte) — qualité préservée, fichiers plus volumineux
  • FLAC (conteneur CAF, sans perte) — compression sans perte au format Core Audio
  • PCM linéaire (WAV, non compressé) — compatibilité maximale pour le montage

L’extension est choisie automatiquement (.m4a, .caf ou .wav).

★

Save Location / Emplacement d’enregistrement

English

Between Export Format and Controls, the app shows where files are saved — inside the app’s sandbox folder under Library/Application Support/. Each recording gets a timestamped filename. Use Open Recordings Folder (in Playback) to open the folder in Finder.

Français

Entre le format d’exportation et les commandes, l’application indique où les fichiers sont enregistrés — dans le dossier sandbox de l’app, sous Bibliothèque/Application Support/. Chaque enregistrement reçoit un nom horodaté. Utilisez Ouvrir le dossier d’enregistrements (section Lecture) pour l’ouvrir dans le Finder.

3

Controls / Commandes

English

Manage the active recording session.

  • Start Recording (⌘R) — begins capture after permissions and source setup
  • Pause / Resume — temporarily stops writing audio; elapsed time excludes paused intervals
  • Stop (⌘.) — finishes the file and saves it

Status row: red waveform icon, elapsed time (e.g. 42:58), and Input level percentage.

Level meter: horizontal bar showing live audio level (roughly −55 dB to 0 dB). If the bar stays empty, check the source, volume, and permissions.

Français

Gérez la session d’enregistrement en cours.

  • Démarrer l’enregistrement (⌘R) — démarre la capture après autorisation et configuration de la source
  • Pause / Reprendre — suspend l’écriture audio ; le temps écoulé exclut les pauses
  • Arrêter (⌘.) — finalise et enregistre le fichier

Barre d’état : icône de forme d’onde rouge, durée écoulée (p. ex. 42:58) et pourcentage de niveau d’entrée.

Indicateur de niveau : barre horizontale montrant le niveau audio en direct (environ −55 dB à 0 dB). Si la barre reste vide, vérifiez la source, le volume et les autorisations.

4

Playback / Lecture

English

Listen to recordings and manage files without leaving the app.

  • Choose File… — pick any supported audio file on disk
  • Play (⌘↩) — start playback
  • Pause (⌘/) — pause playback
  • Stop (⇧⌘S) — stop and return to the start
  • Play / Continue (Space) — toggle play and pause
  • Reveal in Finder — show the current file in Finder
  • Open Recordings Folder — open the app’s recordings directory
  • Copy Path — copy the file path to the clipboard

Playback position slider — drag to scrub; release to jump to that time. Current and total duration appear on the left and right.

Last recording shows the path of the most recently saved file.

Français

Écoutez vos enregistrements et gérez les fichiers sans quitter l’application.

  • Choisir un fichier… — sélectionner un fichier audio pris en charge
  • Lecture (⌘↩) — démarrer la lecture
  • Pause (⌘/) — mettre en pause
  • Arrêter (⇧⌘S) — arrêter et revenir au début
  • Lecture / Continuer (Espace) — basculer lecture / pause
  • Afficher dans le Finder — localiser le fichier actuel
  • Ouvrir le dossier d’enregistrements — ouvrir le répertoire des enregistrements
  • Copier le chemin — copier le chemin dans le presse-papiers

Curseur Position de lecture — faites glisser pour parcourir ; relâchez pour sauter à ce moment. La durée actuelle et totale s’affichent à gauche et à droite.

Dernier enregistrement affiche le chemin du fichier le plus récent.

macOS Permissions / Autorisations macOS

English

  • Screen Recording — required for display and window capture. Grant in System Settings › Privacy & Security › Screen Recording, then restart the app if prompted.
  • Microphone — required only for 3.5 mm Headphone Jack / Line In. Grant in System Settings › Privacy & Security › Microphone.

Français

  • Enregistrement de l’écran — requis pour la capture d’écran et de fenêtre. Accordez dans Réglages système › Confidentialité et sécurité › Enregistrement de l’écran, puis relancez l’app si nécessaire.
  • Microphone — requis uniquement pour Prise casque 3,5 mm / entrée ligne. Accordez dans Réglages système › Confidentialité et sécurité › Microphone.

Quick Start / Démarrage rapide

English

  1. Choose a source and target display/window if needed.
  2. Pick an export format.
  3. Click Start Recording and confirm the level meter moves.
  4. Click Stop when finished.
  5. Use Playback to review, or Open Recordings Folder to share the file.

Français

  1. Choisissez une source et l’écran ou la fenêtre cible si nécessaire.
  2. Sélectionnez un format d’exportation.
  3. Cliquez sur Démarrer l’enregistrement et vérifiez que l’indicateur de niveau bouge.
  4. Cliquez sur Arrêter une fois terminé.
  5. Utilisez la section Lecture pour réécouter, ou Ouvrir le dossier d’enregistrements pour partager le fichier.
